## Features
- 구현기능 1: Js로 공공 데이터포털 제공api 활용하여 기상정보 및 대기환경 정보 파싱
- 구현기능 2: jQeury 를 활용하여 게시판 탭 이동 기능 구현
- 구현기능 3: bx slider 활용하여 캐로셀 기능 구현
- 구현기능 4: 구글 애널리틱스적용하여 방문자의 데이터 웹 로그 분석
